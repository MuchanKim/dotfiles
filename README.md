# dotfiles

Personal development environment configuration.

## Setup

```bash
git clone git@github.com:MuchanKim/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## What's included

| Tool | Files | Description |
|------|-------|-------------|
| Claude Code | `claude/CLAUDE.md` | Global agent instructions |
| Claude Code | `claude/rules/` | Modular rule files (git, obsidian, apple-platform) |
